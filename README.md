# Schedule

## 排班流程
- 先算出每個月的總點數
- 由資歷低的先算7, 6, 5, 4 點, 
- 如果超過，再從資深的開始減點,最低3點 
- 點數分配完後開始排班
- 每個人預不要什麼班就不要排
- 一個一個人去排入
- 不要有出現QOD的情況
- 假日最多兩班
- 星期五輪流上
- 如果點數不平均，則這個月記錄下來超過的點數，下個月再減回來
